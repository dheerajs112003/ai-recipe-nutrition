AI Recipe & Nutrition App

An AI-powered app that generates recipes and nutritional information
based on user-provided ingredients.

 Tech Stack
- Next.js
- React
- OpenAI API
- Tailwind CSS

 Features
- Ingredient-based recipe generation
- Nutrition breakdown
- Clean UI

Setup
npm install  
npm run dev
create .env 
add OPENROUTER_API_KEY=sk-or-v1-...
